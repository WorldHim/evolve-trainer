#### 使用说明

1.  导出存档后将存档文件复制到 待修改区
2.  点击 解析存档 ，再点击数据修改按钮
3.	修改后存档显示在 已修改区 ，数据变更显示在 日志

#### 以下为按钮功能

5.	增加有限资源 - 已开放且有上限(上限>10k)资源 加至 资源上限-10k
6.	增加无限资源 - 已开放且无上限资源 增加 9.99G
6.	增加灵魂宝石 - 灵魂宝石 增加 999
7.	清空日志 - 清理日志
8.	存入待修改区 - 将已修改区的数据存入待修改区，点击该按钮后可再次点击数据修改按钮进行二次修改
